Usage

Colors should have semantic value when used in interfaces. Stick to the following rules:

  • Use green for “Go” actions like “Sign Up” or “Send”.
  • Use red for “Danger” actions like deleting.
  • Use yellow for “Waiting” actions like loading, warnings, or non-critical errors.
