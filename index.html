TECH:ETHICS Conference Style Guide

Welcome to your Tech: Ethics Design System Guideline. This guideline a visual language where you can find all the company’s design elements for different usage.

About Tech: Ethics

Tech: Ethics is a global conference that take place in Tokyo, Japan. This conference is going to address the ethical problems that we see in the conveniency of technology.

Goal

Our target audiences are going to be the young generation, 18 to 40 years old. The goal of our design is to create a fun yet serious feelings that attract audience to learn more.

Usage

Colors should have semantic value when used in interfaces. Stick to the following rules:

  • Use green for “Go” actions like “Sign Up” or “Send”.
  • Use red for “Danger” actions like deleting.
  • Use yellow for “Waiting” actions like loading, warnings, or non-critical errors.
